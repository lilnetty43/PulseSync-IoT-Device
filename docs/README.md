# Project Documentation

This directory contains the final documentation for the **PulseSync IoT** project, including the written report and presentation materials submitted for CPEG/ELEG298.

## Contents

### Final Report
- `report/PulseSync_IoT_Final_Report.pdf`  
  Comprehensive technical report covering system design, hardware implementation, firmware development, testing, and results.

### Presentation Slides
- `slides/PulseSync_IoT_Final_Presentation.pdf`  
  Final project presentation summarizing objectives, design approach, challenges, and outcomes.

## Related Directories
- Hardware design files are located in `/hardware`
- Firmware source code is located in `/firmware`
- PCB images and placement diagrams are located in `/images`
